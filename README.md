# Telegram Notification Bot

Бот для отправки уведомлений каждые 15 минут в личку Telegram.

## Установка

```bash
pip install python-telegram-bot python-dotenv schedule
```

## Запуск

1. Создай `.env` файл:
```
TELEGRAM_BOT_TOKEN=твой_токен_бота
```

2. Запусти бота:
```bash
python bot.py
```

3. Отправь боту команду `/start`
